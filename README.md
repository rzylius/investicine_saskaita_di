Čia pateiktos Claude Code instrukcijos Interactive brokers ir Swedbank investicinės sąskaitos pavertimui į 
csv failą, kurė priima VMI deklaruojant investicinę sąskaitą.

Šis kodas skaito Interactive brokers html failus:
- IB atsisiųskite activity report 2024-12-31 (day report) ir
- IB annual activity report 2025

Ataskaitas dėkite į "Source". 

Promptas sugeneruos failą įkėlimui, ir kitą failą tokį patį su anotacijomis (kur paaiškinta kiekviena tranzakcija - svarbu pasitikrinimui).
